# Simulador de Investimentos em Fundos Imobiliários - Excel
## Descrição do Desafio
Contruir uma planilha e aplicar os conceitos de Excel para desenvolver um Simulador de investimentos em FII (Fundos de Inestimentos Imobiliários). 
## Objetivos do Desafio
Automatizar cálculos e realizar projeções futuras relacionadas ao Rendimento Mensal e Dividendos, tendo como base o perfil de cada cliente, além de responder perguntas dos investidores como tempo e valor necessários de investimento, possibilitando saber o patrimônio acumulado.
## Aprendizado
Criar Simulador de Investimentos utilizando o Excel; Aplicar cálculos financeiros e fórmulas complexas; Planejamento Financeiro; Como funciona o FII. 
## Intruções de Uso
- No Excel, abra o arquivo;
- Defina as Configurações: salário do Cliente e Rendimento da Carteira;
- Configure o Investimento Inicial: qual o valor que o Cliente deseja investir, por quantos anos e a taxa de Rendimento Mensal.
- A partir desses dados, a ferramenta, automaticamente, fará os cálculos de Patrimônio Acumulado e Dividendos.
